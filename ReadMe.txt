== Description ==

You need to be a registered member of SMSGatewayCenter.com to use this plugin.
SGCSMS allows you to send bulk sms from your wp site.
Settings: Type your username and password in settings.
Sender Name: It has to be approved by SMSGatewayCenter.com Team and it can be alpha-numeric and also maximum of 6 characters long.
Mobile: You can include comma separated mobile numbers and can send upto 150 multiple recipients at once.
Message: Currently we are only allowing English SMS message from wp sites.
Delivery Report: You can view your sent SMS report in this section.
Short Code Report: You can view your incoming SMS report in this section.

== Installation ==

= AUTOMATIC INSTALLATION =

1. Log in to your WordPress blog and visit Plugins->Add New.
2. Search for SGCSMS Bulk SMS Sender, click "Install Now" and then Activate the Plugin
3. Visit the "SGCSMS -> Settings" menu.
4. Configure the options with your registered username and password
5. Visit the "SGCSMS -> Send Bulk SMS" menu to send bulk SMS.
6. Manually send bulk SMS.


= MANUAL INSTALLATION =

1. Copy the entire /sgcsms-bulk-sms-sender/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. Visit the "SGCSMS -> Settings" menu.
4. Configure the options with your registered username and password
5. Visit the "SGCSMS -> Send Bulk SMS" menu to send bulk SMS.
6. Manually send bulk SMS.

== Frequently Asked Questions ==

Where can I find my Username and Password?
You need to be registered member with www.smsgatewaycenter.com and your account status needs to be active in order to work with this plugin.

How should I use mobile numbers to To column?
You need to use 10, 11 or 12 digits Indian mobile number as 09999999999 or 9999999999 or 919999999999.

Can I send to multiple mobile numbers?
Yes, you can, just comma separate the mobile numbers as 9999999991,919999999992, 09999999993

Is there any limit while sending bulk SMS?
Actually no, but we recommend you to use maximum 150 mobile numbers in To field.

Is there any limit on Incoming SMS report?
No, We support 30 days retrieval and you can see all those 30 days report.

Does any of the information gets inserted in the database?
No, We do not record any data in your database.

What should I use in settings page?
Just use your registered username and password in the settings page.
